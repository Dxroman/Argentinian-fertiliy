# Argentinian-fertiliy
Description of the current fertility situation in Argentina, from economic and psychological perspective and estimations on its future.  


In this folder i include a spanish version of this work (original). An english .ipynb version will be posted very soon. Since this original work was done using Excel, the only database available for the .ipynb file is already filtered, so the work in this enviroment is purely statistical. Both versions in both languajes will be available at this repository, in order to offer two languages choices, coding and statistical skills, as well as economic insight.
